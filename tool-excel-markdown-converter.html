Excel ⇄ Markdown 双向转换工具

支持 Excel 粘贴 → Markdown 表格，也支持 Markdown 表格 → Excel 格式（TSV/复制可用）

输入源
输入变化后自动转换为右侧格式
转换结果
根据左侧模式和右侧目标格式自动转换
双向转换说明
① Excel → Markdown
左侧选择「Excel 格式」，粘贴从Excel复制的表格内容（制表符分隔，含表头行），右侧选择「Markdown 表格」，自动生成标准MD表格。
② Markdown → Excel
左侧选择「Markdown 表格」，粘贴完整MD表格（如 | 列A | 列B |
|---| --- |），右侧选择「Excel 格式」，自动转为制表符分隔的文本，可直接复制到Excel中粘贴为单元格。
③ Excel ↔ Excel
左侧Excel模式，右侧Excel模式 = 格式校验/清洗（保留原始tsv）。
④ MD ↔ MD
左侧MD模式，右侧MD模式 = 标准化Markdown表格排版。
