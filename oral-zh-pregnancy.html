Pregnancy is a hardcore sci-fi epic // of war, // peace // and compromise.
怀孕是一部关于战争、//和平//和妥协的硬核科幻史诗//。

Babies are basically very cute tumors // that grow a completely new organ // inside a different human.
婴儿基本上是非常可爱的肿瘤，// 在另一个人体内长出一个全新的器官 // 。

Their mom’s body graciously allows this, // while the new life saps her resources // and challenges her immune system to its limits for months.
他们妈妈的身体慷慨地允许这样做，// 而新生命却消耗了她的资源//并在几个月的时间里挑战她的免疫系统到极限 。

Biology is brutal // and beautiful // and metal.
生物学是残酷的//和美丽的//和金属 。

Let us zoom in and observe this brutal struggle for survival // – that in the best case // ends in one of the most magical things: // A new life.
让我们放大并观察这场残酷的生存斗争 // – 在最好的情况下 // 以最神奇的事情之一结束： // 新生活。

Our story begins with an army of tens of millions of sperm cells // that immediately face death.
我们的故事始于一支由数千万个精子细胞组成的大军//它们立即面临死亡 。

They are on a tight timer // before they run out of energy and die.
他们处于一个紧迫的计时器上//，在他们耗尽能量并死亡之前。

But this is the least of their problems right now.
但这是他们目前最不关心的问题 。

Like all bodily openings, // the female reproductive tract is a fortress // defending against invaders.
像所有身体开口一样，//女性生殖道是防御入侵者的堡垒 //。

The future mom’s body has set up deadly barriers // and puts the sperm through a brutal process // that kills almost all of them // and ideally selects the strongest and fittest one.
未来妈妈的身体设置了致命的屏障//，并让精子经历一个残酷的过程//几乎杀死了所有精子//理想情况下，选择最强壮和最健康的精子。

The first deadly obstacle is a highly acidic environment // guarded by hundreds of thousands of guard cells.
第一个致命障碍是高酸性环境 //由数十万个守卫细胞守护。

The acidity kills millions, // especially of the weaker sperm // within the first half hour, // although they have a bit of protection: // Seminal fluid is alkaline // and so it makes this first obstacle a little less deadly.
酸度杀死了数百万人，// 尤其是较弱的精子 // 在前半小时内 ，// 尽管它们有一些保护： // 是碱性的 // 所以它使第一个障碍不那么致命。

The survivors reach the next obstacle, // a treacherous maze // full of protein nets // where millions more find a brutal end // getting stuck and lost.
幸存者到达下一个障碍，//一个危险的迷宫//充满蛋白质网//数百万人在那里找到了残酷的结局//被困住并迷路了。

Now if a woman is in the part of her menstrual cycle // where she is ovulating, // her body helps the sperm out a bit, // by releasing chemicals that guide the way // and by being a bit less hostile.
现在，如果一个女人处于月经周期的//排卵期，//她的身体会帮助精子排出一点 ，//通过释放引导方向的化学物质//和不那么敌对。

Luckily for our brave sperm // this is the case today.
幸运的是 ，我们勇敢的精子 // 今天就是这种情况。

Only a couple hundred // – less than 0.0001% of the original sperm – // pass through into the uterine cavity, // where two tunnels branch out.
只有几百个 // – 不到原始精子的 0.0001% – // 进入子宫腔 ，// 两条隧道分支出来。

Here mucus helps transport the stronger sperm to move on, // while weeding out the weaker ones.
在这里 ，粘液有助于运输较强的精子继续前进，// 同时淘汰较弱的精子。

The mom’s immune cells pick off and devour every cell // that trails behind or takes the wrong turn.
妈妈的免疫细胞会挑选并吞噬每一个落后或走错路的细胞。

And then finally, // maybe just a few dozen sperm reach their goal, // the mighty, // giant egg.
最后 ，//也许只有几十个精子达到了他们的目标，//强大的，//巨大的卵子。

10,000 times bigger than them, // it is an industrial scale monster // loaded with nutrients // and nearly 100,000 mitochondria, // the powerhouse of the cell // – 50 times more than an average cell.
比它们大 10,000 倍，// 它是一个工业规模的怪物 // 富含营养物质 // 和近 100,000 个线粒体，// 细胞的动力源 // – 是普通细胞的 50 倍。

But the egg is not easily conquered // – the survivors get closer and try to get in // until the egg decides to accept a single one of them.
但蛋并不容易被征服 // – 幸存者靠近并试图进入 // 直到蛋决定接受其中一个。

The genes of two individuals merge.
两个人的基因融合在一起。

The egg stops being part of its mother // – it is something else now. // A new being with its own agenda.
蛋不再是母亲的一部分 // 它变成了别的东西。一个有自己议程的新生命。

Not a human yet, // but the potential to become one. // Like a thought that is not yet a word.
还不是人类，// 但有可能成为人类 。就像一个还没有一个词的想法。

But its mom’s body hasn’t decided yet // if she wants to say it out loud. // It will have to fight for that.
但它妈妈的身体还没有决定//她是否想大声说出来 。它必须为此而战 。

For a few days // it rapidly divides and grows // to a couple of hundred cells, // while traveling down to the uterus, // the part of the female reproductive system // where it will try to make its new home.
几天后 // 它迅速分裂并生长 // 到几百个细胞，// 在向下移动到子宫时，// 女性生殖系统的部分 // 它将尝试在那里建立新家 。

Here it begins to divide into two specialised teams of cells // - one will eventually become the baby.
在这里 ，它开始分裂成两个专门的细胞小组//-一个最终将成为婴儿。

The other cells are trophoblasts // and their job is to turn into a temporary organ inside the mother: // the placenta, // that will make pregnancy possible and eventually die.
其他细胞是滋养层 // 它们的工作是变成母亲体内的临时器官： // 胎盘 ，// 这将使怀孕成为可能并最终死亡。

This means that when you were an embryo // there was this whole other part of you, // not like a twin, // even closer, // maybe like a guardian angel clone.
这意味着当你还是胚胎时，// 你体内有另一部分，// 不像双胞胎，// 更接近，// 也许像守护天使的克隆体。

These cells never had the chance to become a human, // their only purpose in life was to make you exist.
这些细胞从未有机会成为人类，// 它们生命的唯一目的就是让你存在 。

In any case, // the following days are the most dangerous // for the potential new life.
无论如何 ，接下来的几天对潜在的新生命来说是最危险的。

Like a tennis ball rolling through syrup // the young embryo moves along the uterus // and tries to take a hold.
就像网球在糖浆中滚动一样 // 幼小的胚胎沿着子宫移动 // 试图抓住它。

An intense chemical dialog between two living things begins, // the embryo releases dozens of chemicals // that announce its presence // and asks the guard cells of the uterine wall // to allow it to please, please, please attach itself // so it can survive.
两个生命体之间开始激烈的化学对话，// 胚胎释放数十种化学物质 // 宣告其存在 // 并请求子宫壁的守护细胞 // 允许它 // 求你，求你附着 // 以便生存。

The uterus responds with dozens of hormones and immune signals itself.
子宫本身会通过数十种激素和免疫信号做出反应。

If it is satisfied with the quality of the chemical conversation // its cells allow it. // If it doesn’t // the embryo will be lost and die.
如果它对化学对话的质量感到满意 // 它的细胞会允许它。如果不满意 // 胚胎将会丢失并死亡。

This is where we begin to see // that the interests of both living beings no longer completely align anymore.
这时我们开始看到 // 两个生命的利益不再完全一致 。

Pregnancy is a huge energy investment // so the mother’s body will eject the embryo // if it doesn’t see it as viable.
怀孕是巨大的能量投入 // 所以如果母亲的身体认为胚胎不可存活 ，会把它排出去 //

The embryo on the other hand // faces a life and death situation // and tries to stay alive at any price.
而胚胎则面临生死关头 ，不惜一切代价求生存。

So it’s not asking politely // but deploying thousands of infiltration units: // Bubbles filled with genetic material, // kind of like a human virus!
所以这不是礼貌地请求 // 而是部署数千个渗透单位 ：// 充满基因物质的泡泡，// 有点像人类病毒！

They sort of invade the uterine cells // and try to brainwash them, // so they help it attach itself, // rather than rejecting it.
它们会侵入子宫细胞 // 并试图洗脑它们，// 所以它们帮助子宫附着，// 而不是排斥。

During this time // one of the weirdest features of pregnancy occurs: // the uterus provides uterine milk, // not real milk but a clear fluid filled with nutrients and hormones // that the young embryo sucks up hungrily // to gain additional energy until it gets inside.
在这段时间里 // 怀孕中最奇怪的现象之一发生了：// 子宫分泌的乳汁 ，// 不是真正的乳汁，而是一种充满营养和激素的透明液体 // 年轻胚胎贪婪地吸取它 // 以获得额外能量直到进入体内。

If the embryo is allowed in, // the implantation was successful, // the first critical hurdle.
如果胚胎被允许进入，// 着床成功，// 第一个关键障碍 。

The next step is to reach the blood vessels of its mother, // the only way to survive // if it doesn't want to starve.
下一步是到达母亲的血管 ，// 这是唯一的生存之道 // 如果它不想饿死的话。

The trophoblasts begin to clone themselves on a massive scale // and turn into different specialists.
滋养层细胞开始大规模克隆自己 // 并转变为不同的专家。

One of them turns into a violent invader // that starts drilling into the uterine tissue // like a tiny parasitic octopus // with many arms, // spreading and growing.
其中一个变成了暴力入侵者 // 开始钻入子宫组织 // 像一只有许多手臂的小寄生章鱼 // 扩散和生长。

This is a brutal process, // ordering loads of it’s mom’s uterus cells to destroy themselves, // killing others directly // or even devouring some cells whole.
这是一个残酷的过程，// 命令大量母亲的子宫细胞自我毁灭，// 直接杀死其他细胞 // 甚至整颗细胞被吞噬。

While this sounds brutal, // and it is, // this process is highly regulated // and doesn’t hurt the mom.
虽然听起来很残酷 ，// 确实如此，// 这个过程非常严格， 且不会伤害妈妈。

It is another test. // The mom’s body monitors carefully // how the embryo is doing // – is it growing quickly or if it is more chill.
这又是一次测试。妈妈的身体会仔细监测胚胎的状况 // ——是快速生长还是比较冷静 。

If an embryo has genetic damage or chromosomal abnormalities // it will spend way more energy on repairing itself // and maybe grow more erratically.
如果胚胎有遗传损伤或染色体异常 // 它会花费更多能量修复自己 // 可能生长得更不规律 。

This makes it metabolically noisy, // releasing loads of chemicals the mom’s immune cells pick up.
这会让它代谢上很吵 ，// 释放大量化学物质，妈妈的免疫细胞会捕捉到这些化学物质。

Which provokes them and makes it more likely // that they will destroy it.
这激怒了他们，也让他们更有可能摧毁它。

On the other hand, // if the embryo is weak // it is metabolically too quiet // and the mom will stop talking to it // – which also ends the pregnancy.
另一方面 ，// 如果胚胎虚弱 // 它代谢过于安静 // 妈妈会停止与胚胎交流 // ——这也会终止妊娠。

This embryo is just right, // so the mom’s cells release a flood of different chemicals // to help it out, // support it growing // and most importantly, // activate her immune system.
这个胚胎正合适，// 所以妈妈的细胞会释放大量不同的化学物质 // 来帮助它生长 ，// 支持它生长 // 最重要的是，// 激活她的免疫系统。

Usually this would be very bad // – the immune system kills everything that is not part of the mom’s body, // and this embryo is clearly not part of her.
通常这会很糟糕 // 免疫系统会杀死所有不属于妈妈身体的部分，// 而这个胚胎显然不是她的一部分。

But the immune cells of the uterus surround the embryo // and start helping it, // guiding the trophoblast to grow further.
但子宫的免疫细胞包围着胚胎 // 开始帮助它，// 引导养层细胞继续生长。

They create a physical and chemical safe zone // that tells dangerous immune cells like T Cells to stay away.
它们会创造一个物理和化学的安全区 // 告诉危险的免疫细胞如 T 细胞远离 。

The embryo has its own motives // and doesn't want to rely on its mothers goodwill alone // – so defensive trophoblasts send out signals // that kill her immune cells // if they get too close // and could start attacking it.
胚胎有自己的动机 // 不愿仅依赖母亲的善意 // ——因此防御性营养层细胞会发出信号 // 如果靠近会杀死她的免疫细胞 // 并可能开始攻击。

Hundreds of cells go even further // and leave the embryo behind.
数百个细胞甚至更进一步 // 并留下胚胎 。

They spread all across the mom’s body, // entering organs, // even the brain.
这些病会扩散到妈妈全身，// 进入器官，甚至进入大脑。

We don’t know what all of these cells are doing // but we think that they are probably telling her immune system // that the embryo is not to be attacked. // That it should be left alone or even protected.
我们不知道这些细胞都在做什么 // 但我们认为它们很可能是在告诉她的免疫系统 // 胚胎不应被攻击。应该让它被单独处理 ，甚至被保护。

These cells may stay inside the mother // for years or even decades. // It is likely that parts of you // are still in your mother.
这些细胞可能会在母亲体内停留多年甚至数十年 。你身体的部分很可能仍在母亲体内。

Around 8 weeks after the egg was fertilized, // the transition from the embryo to fetus begins // – the size of an olive, // its organs begin to form // and it turns from a blob // into something vaguely human-like.
卵子受精后大约 8 周 ，// 胚胎到胎儿的过渡开始 // 橄榄大小，// 其器官开始形成 // 从一团团变成模糊的人形。

Is there a clear point // where a clump of cells becomes a human? // Not really.
有没有一个明确的节点 //，一团细胞会变成人类？ 其实没有 。

It is a fluid transition // and every society and person // – be it morally or legally // – marks this moment differently.
这是一个流动的过渡 ///每个社会和每个人/无论是道德上还是法律上 /——都会以不同的方式标记这一刻。

Meanwhile // construction trophoblasts are busy building a spongy fingerlike structure // that expands further into the uterus, // a completely new organ: // The placenta.
与此同时 ，// 建设营养层细胞正忙着构建一个海绵状的指状结构 // 向子宫更深处扩展，// 一个全新的器官：胎盘。

An organ that you once grew inside your mother, // that was ejected after your birth // and died a silent death // while everybody was busy welcoming you into the world.
那是你曾经在母亲体内生长的器官，// 在你出生后被弹出 // 默默死去 // 而所有人都忙着迎接你来到这个世界。

In our story // the placenta is now the new home of the fetus. // An enormous fortress, // protecting it from microbes that could infect and kill it // in its still pretty fragile state.
在我们的故事中 // 胎盘现在是胎儿的新家。一座巨大的堡垒，// 保护胎盘免受可能感染并杀死胎儿的微生物侵害 // 胎儿仍然脆弱的状态。

It even has its own mini immune system, // placental immune cells that gobble up anything that poses a threat.
它甚至有自己的小型免疫系统，// 胎盘免疫细胞，吞噬任何威胁 。

Other placental cells creep along the inside of the mom's blood vessels, // stretching them and connecting the fetus // through the umbilical cord on the other side.
其他胎盘细胞沿着母亲血管内部蠕动，拉伸血管，并通过脐带连接胎儿。

With the blood flow secured, // it’s time to load it with as much food as possible.
血液流动稳定后，// 是时候尽可能多地给它装满食物了。

The placenta releases hormones that funnel glucose directly to the fetus, // stealing energy from its mom.
胎盘释放激素，将葡萄糖直接引导给胎儿，// 从母体内偷取能量。

If the fetus goes too far and asks too much, // this can sometimes lead to gestational diabetes for the mom // during the pregnancy, // starving her body of energy.
如果胎儿要求过多，// 这有时会导致孕期妊娠糖尿病 ，// 导致身体缺乏能量。

The mothers body is trying to support the new life, // but not at the cost of her own survival.
母亲的身体试图支持新生命，但不愿以牺牲自身生存为代价。

In a sense // the mothers genes inside the fetus // still have a stronger allegiance to her // than to the new being – but its fathers genes don’t.
从某种意义上说 ，// 胎儿体内的母亲基因 // 对她// 的忠诚度比对新生命更强——但父亲的基因则不然。

They want the fetus to survive at all costs.
他们不惜一切代价希望胎儿能存活。

So while there is a sort of fragile peace between both parties, // it is just that // – their interests are not perfectly aligned // and both sides have to deal with that.
所以，虽然双方之间存在某种脆弱的和平 ，// 只是 // 他们的利益并不完全一致 // 双方都必须面对这一点 。

In the next few months // the fetus will increase its weight over a billion times, // which demands a staggering amount of energy and cooperation.
在接下来的几个月里，胎儿的体重将增加十亿倍以上，这需要惊人的能量和配合。

They still need to work together // to get out of this alive.
他们仍然需要合作 // 才能活着脱身 。

But if the fetus gets to this point in the pregnancy, // chances are pretty high that it will become a baby like you once were, // a proper human, // like you are today. // A being with immense potential.
但如果胎儿怀孕到这个阶段，// 很可能会变成像你曾经那样的婴儿，// 成为一个真正的人类，// 像你今天这样。一个拥有巨大潜力的存在。

If you are alive today // then you went through an amazingly brutal selection process.
如果你今天还活着 // 那你经历了极其残酷的选拔过程 。

From a little pack of genes // traveling through an incredibly deadly obstacle course, // fighting for your survival, // with desperate words spoken through chemicals // and sneaky actions trying to outsmart the system, // fighting to be alive in this world.
从一小群基因 // 穿越极其致命的障碍赛 ，// 为生存而战，// 用化学物质说出绝望的话语 // 偷偷摸摸地试图智胜系统，// 在这个世界里挣扎求生。

But this is only one side of the story // – you were also chosen – // by hormones that led you to the right place, // by the egg allowing you to merge into one.
但这只是故事的一面 // ——你也是被选中的 ——// 被引导你到正确的地方的荷尔蒙，// 被卵子让你融合成一个。

By your mom’s body that liked how you spoke to her, // by her cells that protected and cared for you. // Nourished by an incredible amount of energy // that your mother gifted you.
被妈喜欢你和她说话的方式所滋养，// 被她保护和关心你的细胞所滋养。被你母亲赐予的惊人能量//滋养。

Our biology is a brutal, // unforgiving, // but necessary part of the greatest wonder there is: // The creation of new life, // of you, // of all of us.
我们的生物学是残酷、 无情的，但却是最伟大奇迹中不可或缺的一部分：创造新生命，创造你，创造我们所有人。

Kids fill us with a sense of wonder // – it’s our job to foster this sense of wonder in them as well.
孩子们让我们充满了惊奇感 // ——我们的工作就是培养他们内心的这种好奇心。

Our sponsor KiwiCo is a great start // to spark curiosity in your kids // from the earliest moments on // – basically from birth.
我们的赞助商 KiwiCo 是一个很好的起点 // 从孩子出生起 // 从最早的时刻起激发他们的好奇心

This is one of their crates for newborns, // so let’s explore it together:
这是他们给新生儿准备的一个笼子 ，// 让我们一起探索看看：

There is a tummy time pillow … // some black and white cards.. // a sensory mat – and a mirror?
有趴着睡枕......// 一些黑白卡片......感官垫——还有镜子？

It’s the Let’s Bond! Panda crate, // a kit that helps your newborn explore its brand-new sensations of vision, // hearing // and motor control.
这就是“让我们联结”！熊猫笼，// 一个帮助新生儿探索全新视觉、 听觉和运动控制感官的套件。

At the same time, // you get to know this tiny new human in your life // and learn about how it currently experiences the world.
与此同时 ，// 你会认识这个生命中这个小小的新生命 // 并了解他目前如何体验这个世界。

This crate is part of the KiwiCo Panda Crate line // of research-backed toys and activities // to support your baby’s development.
这个笼子是 KiwiCo 熊猫笼系列的一部分，// 有研究支持的玩具和活动，支持宝宝的成长。

You can promote vision, // hearing // and gross motor skills in the first months of a baby’s life // or rhythm and language skills later.
你可以在宝宝生命的最初几个月促进视力、听觉和大运动技能 ，或者以后培养节奏和语言技能。
