主线一：打车核心盘受损
表象：漏斗首环折损
打车完单整体下降 -0.22%，且流失并非发生在比价阶段，而是在「首页冒泡」即遭遇断崖。
流失人群集中在核心高价值用户：近30天21单+、安卓高端机客群。
分析师诊断：DID 锁定根因
通过 DID 因果推断，将负向精准锁定于「访问过货运顶Tab」的人群，剥离其他干扰。该群体贡献了几乎全部负向（约2.5万单）。
行为下钻 1 - 动线“迷路”：用户在货运页频繁点击【首页】底导试图返回，但因机制变更被困，阻断发单。
行为下钻 2 - 认知“误触”：约 10% 用户在货运地址栏操作后2分钟内退回打车冒泡，证实 UI 过度相似导致混淆。
策略落地：快速止血与迭代
1. UI 认知隔离设计 已落地
针对误触混淆，产品配置上线 颜色差异化设计（包含顶 Tab 文字、Banner、搜索框的视觉隔离），帮助用户快速识别频道。
2. 修复底层动线断点 4.16 灰度上线
针对迷路问题，在底导新增 一键返回态。当处于货运等非出行 Tab 时，点击【首页】立刻重置回打车主页，恢复肌肉记忆。
3. 流量弹窗错峰与降噪 已落地
由于引导弹窗放大了误触基数，运营已调整为 错峰投放，彻底避开网约车早晚高峰；二期规划基于连点行为的精准触发。
主线二：货运流量假繁荣
表象：DAU涨，交易平
货运 DAU 猛增 +38%，但平台完单量未见同步提升。
货运顶导 CTR (1.12%) 实际上略低于原先的底导 (1.42%)，品类自然渗透率不及预期。
分析师诊断：人群错配
80% 增量是由引导弹窗强拽入的“潜新用户”（逛一逛心态），缺乏转化动机。
核心老客并未向顶导迁移（迁移率由 65% 降至 49%），反而大量沉淀在金刚位（动线更符合原有习惯）。
当前频道页仍沿用 B 端交易动线，导致 C 端用户对车型认知成本过高，漏斗卡死。
策略落地：B/C分层精细化
1. 人群流量精细分发 已落地
顺应自然选择，实施人群隔离导流：老客核心导向金刚位保障交易底盘，潜新导向顶 Tab 培育心智。
2. C端动线与心智重构 AB 验证中
基于 C 端痛点，规划 简化车型下单动线。联动品牌侧测试将 Tab 名称更迭为「快送」，强化 C 端小件/即时心智。
