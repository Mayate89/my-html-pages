Terrorism is very scary, // especially when it happens close to home // and not in some faraway place.
恐怖主义非常可怕，//尤其是当它发生在离家很近的地方//而不是在某个遥远的地方时。

Nobody likes to be afraid, // and we were eager to make the fear go away.
没有人喜欢害怕，// 我们渴望让恐惧消失 。

So // we demanded more security.
所以 // 我们要求更多的安全性。

In the last decade, // it’s become increasingly normal // for civil liberties to be eroded // and for government agencies to spy on citizens, // to collect and store their personal information.
在过去十年里，// 公民自由被侵蚀变得越来越常见 // 政府机构监视公民，// 收集和存储他们的个人信息。

Regardless of whether you’re a fan of right- or left-wing policies, // this affects every one of us.
无论你是右翼政策还是左翼政策的拥护者，// 这会影响我们每个人 。

So we have to take a look at the data // and ask ourselves honestly, // “Has all of this actually made us safer?”
所以我们必须看看数据 // 并诚实地问自己，// “这一切真的让我们更安全了吗？

In the aftermath of 9/11, // the US government concluded that the law had not kept pace with technology.
9/11 事件发生后 ，// 美国政府得出结论，该法律没有跟上技术的步伐 。

It created the Terrorist Surveillance Program // initially to intercept communications linked to al-Qaeda.
它创建了恐怖分子监视计划 // 最初是为了拦截与基地组织有关的通信。

Officials were confident // that if the program had been in place before 9/11, // the hijackers could have been stopped.
官员们相信 // 如果该计划在 9/11 之前就已经实施，// 劫机者本可以被阻止 。

But soon // the new powers were also used to prove guilt by association.
但很快 // 新权力也被用来通过关联来证明有罪 。

The FBI used immigration records // to identify Arab and Muslim foreign nationals in the US.
联邦调查局利用移民记录//识别了在美国境内的阿拉伯和穆斯林外国公民。

On this basis, // 80,000 individuals were required to register, // another 8,000 were called in for FBI interviews, // and more than 5,000 locked up in preventive detention.
在此基础上 ，// 80,000 人被要求登记，// 另有 8,000 人被传唤接受联邦调查局采访，// 5,000 多人被关在预防性拘留中。

Not one terrorist was found // in what’s been called // the most aggressive national campaign of ethnic profiling // since World War II.
在所谓的//自二战以来最激进的全国性种族定性运动中，没有发现一个恐怖分子//。

How commonplace it’s since become // for government agencies to collect and store the personal data of citizens // was made plain by the leak of the Snowden documents in 2013.
从那以后，政府机构收集和存储公民个人数据变得多么普遍 //2013 年斯诺登文件的泄露让人们明白了这一点。

They showed how the NSA can demand information about users // from firms like Microsoft or Google // in addition to their daily collection of data from civilian internet traffic // such as email content and contact lists.
他们展示了美国国家安全局如何要求 Microsoft 或 Google 等公司提供有关用户的信息 // 除了他们每天从民用互联网流量中收集数据 // （例如电子邮件内容和联系人列表）之外。

So, instead of focusing on criminals, // governments are increasingly turning their attention to everyone.
因此， 政府不再关注罪犯，而是越来越多地将注意力转向每个人 。

But if you are looking for a needle in a haystack, // adding more hay to the stack // isn’t going to make it any easier // to find the needle.
但是 ，如果你正在大海捞针，// 在大海捞针堆中添加更多的干草 // 不会让 // 更容易找到针。

On the contrary, // every recent success announced by the NSA // has come from classic targeted surveillance.
相反 ，// 美国国家安全局最近宣布的每一次成功 // 都来自经典的有针对性的监视 。

Despite high hopes, // the NSA surveillance program has not stopped any major terror attack.
尽管寄予厚望，但 // 美国国家安全局的监控计划并没有阻止任何重大恐怖袭击。

For instance, // one of the Boston Marathon bombers was already a target of the FBI.
例如 ，// 波士顿马拉松炸弹袭击者之一已经是联邦调查局的目标。

So // what we need is not even more random data, // but better ways to understand and use the information we have.
因此 ，我们需要的不是更多的随机数据，而是更好的方法来理解和使用我们所拥有的信息。

Spy agencies are also pushing to cripple encryption.
间谍机构也在推动削弱加密 。

In early 2016, // the FBI asked Apple to produce a backdoor program // to disable the encryption of a terrorist’s iPhone.
2016 年初，// 联邦调查局要求苹果公司制作一个后门程序 // 以禁用恐怖分子 iPhone 的加密。

Apple publicly declined, // not only because this tool could be used to permanently weaken the privacy of law-abiding citizens worldwide, //
苹果公开拒绝，// 不仅因为这个工具可能被用来永久削弱全球守法公民的隐私， //

but fearing to open the floodgates for governments requesting access // to a technology used by billions of people, // a fear shared by security experts and cryptographers.
但担心为要求访问 // 数十亿人使用的技术的政府打开闸门 ，// 安全专家和密码学家也有同样的担忧。

A few weeks later, // the FBI revealed that they had hacked the phone themselves, // basically admitting that they lied to the public about the need for a backdoor, //
几周后 ，// 联邦调查局透露他们自己入侵了手机，// 基本上承认他们在需要后门方面向公众撒谎， //

which questions how trustworthy spy agencies are // in the debate about privacy and security, //
这也质疑了间谍机构在隐私和安全辩论中的可信度 //

especially considering that the NSA, for example, // already has the capability to turn on your iPhone microphone // or activate your laptop camera // without you noticing.
尤其是考虑到 NSA，比如说，// 已经有能力在你不注意的情况下打开你的 iPhone 麦克风 // 或激活你的笔记本摄像头 // 。

Concerns about this are often met with the argument, // “If you have nothing to hide, you have nothing to fear.”
对此的担忧常常会被反驳为“//” 如果你无所隐瞒，就无所畏惧 。”

But this reasoning only creates a climate of oppression.
但这种推理只会制造压迫氛围 。

Wanting to keep certain parts of your life private // doesn’t mean you’re doing anything wrong.
想把生活中某些部分保密 // 并不意味着你做错了什么。

Right now, // we live in a democracy. // But imagine the damage // the wrong person could do // with all our data // and such easy access to our devices.
现在 ，// 我们生活在一个民主社会。但想象一下， 错误的人可能会/对我们所有的数据//以及如此轻易访问我们设备的访问造成多大的损害。

Anti-terrorism laws allow the authorities // to investigate and punish non-terrorism-related crimes more aggressively.
反恐法律允许当局更积极地调查和惩罚非恐怖主义相关犯罪。

If you give law enforcement powerful tools, // they will use them.
如果你给执法部门强大的工具，// 他们会使用这些工具。

That’s why democratic oversight is so important: // even if those tools and laws aren’t used against you today, // they might be tomorrow.
这就是为什么民主监督如此重要：// 即使这些工具和法律今天没有被用来对付你，// 明天可能会被用来对付你。

For example, // following the November 2015 Paris attacks, // France expanded its already extensive anti-terrorism laws // by giving law enforcement greater powers // to conduct house raids and place people under house arrest.
例如 ，// 2015 年 11 月巴黎袭击事件后，// 法国扩大了其本已广泛的反恐法律，// 赋予执法部门更大的权力 // 进行房屋突袭和软禁 。

Within weeks, // evidence emerged that these powers were being used for unintended purposes, // such as quashing climate change protests.
几周内 ，// 证据浮出水面，显示这些权力被用于意想不到的目的 ，比如镇压气候变化抗议。

The governments of Spain, // Hungary, // and Poland have introduced more restrictive laws // on the freedom of assembly and speech.
西班牙、匈牙利和波兰政府出台了更严格的集会和言论自由法律。

Freedom of expression and the press in Turkey // has been seriously undermined in the last few years, // with people sentenced to prison // for criticizing the government.
近年来，土耳其的言论和新闻自由受到严重破坏 ，// 有人因批评政府而被判入狱。

None of this is effectively helping us fight terrorism.
这些都没有有效帮助我们打击恐怖主义。

The motivation behind this might be good, // even noble, // but if we let our elected governments limit our personal freedom, // the terrorists are winning.
背后的动机可能是善良的，甚至是高尚的，但如果我们让民选政府限制我们的个人自由，恐怖分子正在获胜 。

What’s worse, // if we’re not careful, // we might slowly move towards a surveillance state.
更糟的是，如果我们不小心，可能会慢慢走向监控状态 。

The data is pretty clear: // the erosion of rights, // along with mass surveillance, // hasn’t led to significant successes so far, // but it has changed the nature of our society.
数据相当明确：// 权利的侵蚀 ，// 伴随着大规模监控 ，// 迄今尚未带来显著成功，// 但它改变了我们社会的本质 。

Terrorism is a complicated problem // without simple solutions.
恐怖主义是一个复杂的问题，没有简单的解决方案。

No security apparatus can prevent a few guys // from building a bomb in their basement.
没有任何安保装置能阻止几个人在地下室制造炸弹。

We should keep the principle of proportionality in mind.
我们应牢记比例原则。

Creating master keys to enter millions of phones // is not the same as // searching a single house.
创建万能钥匙来输入数百万部手机 // 和搜索一栋房子是不一样的。

In most countries, // the law already permits a wide range of actions, // including targeted surveillance.
在大多数国家，// 法律已经允许广泛的行动，// 包括定点监控。

To take full advantage of this existing potential, // we need better international cooperation // and more effective security and foreign policies, // better application of our present laws // instead of new and stricter ones that undermine our freedom.
要充分利用现有潜力，// 我们需要更好的国际合作 // 更有效的安全和外交政策，// 更好地执行现有法律 // 而非削弱我们自由的新法律。

Let us not, // out of fear, // destroy what we are most proud of: // democracy and our fundamental rights and liberties.
让我们不要，// 出于恐惧，// 摧毁我们最引以为傲的东西：// 民主以及我们的基本权利和自由 。
