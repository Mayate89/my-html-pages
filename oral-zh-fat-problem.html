Fat is your most dangerous organ. // Yes, organ.
脂肪是你最危险的器官 。是的，风琴。

While it's often falsely described as a mere expression of laziness or gluttony, // fat is essential for health, // controlling and guiding crucial processes in your body.
虽然脂肪常被错误地描述为懒惰或暴食的表现 ，但// 脂肪对健康至关重要 ，// 控制并引导身体的关键过程。

But if you have too much fat, // it starts to disrupt your metabolism, // becoming one of the most deadly things that can happen to your body.
但如果你脂肪过多，// 它会开始干扰你的新陈代谢 ，// 成为你身体可能发生的最致命的事情之一。

Today, // more people are obese than starving, // which is a huge victory.
如今，肥胖的人比饥饿的人还多，// 这是一个巨大的胜利。

There are all sorts of ideas about the cause of the obesity epidemic, // but it really comes down to one thing.
关于肥胖流行的原因有各种各样的说法，// 但归根结底只有一个。

For millions of years, // humans had to expend a lot of effort to get food // and often faced hunger.
数百万年来，// 人类不得不付出大量努力才能获得食物 // 并且经常面临饥饿。

Our bodies evolved to hold on to every bit of energy.
我们的身体进化出能储存每一丝能量。

And then suddenly // we had an overabundance of food // we didn't need to move much for.
然后突然间 ，我们有了过剩的食物，不需要怎么动。

We made food hyperpalatable // and ultra-processed // with loads of unhealthy fats, // salt // and sugar.
我们让食物变得极其美味 ，// 并且是超级加工的 // 加了很多不健康的脂肪、盐 // 和糖。

Our brains love this, // so our food is extremely hard to resist, // to the point of being addictive.
我们的大脑非常喜欢这个，// 所以我们的食物极难抗拒 ，// 甚至让人上瘾。

It's also extremely convenient, // cheap, // and you get a lot of energy per volume, // so it's not very satiating.
而且非常方便，// 便宜，// 而且每体积能消耗很多能量，// 所以不太让人满足 。

This leads to most of us regularly overeating // without realizing it.
这导致我们大多数人经常暴饮暴食，却浑然不觉。

And food is aggressively marketed, // especially to children.
而且食品营销非常积极 ，尤其是针对儿童。

Fat comes with a lot of shame and blame, // which is pretty unfair // in a world full of things that make you feel nice // and are available within minutes.
肥胖带来很多羞耻和责备，// 在这个充满让你感觉良好的事物 // 几分钟内就能获得的世界中，这相当不公平。

Body fat is also often unfairly villainized // but it's crucially important for your health // and if you don't have enough // you can suffer problems from infertility // to a weak immune system, // fatigue, // mental health issues // and osteoporosis.
体脂也常被不公平地妖魔化 // 但它对你的健康至关重要 // 如果你脂肪不足 // 你可能会遭受不孕不育 、免疫系统薄弱、疲劳 、心理健康问题和骨质疏松等问题。

Still, // none of this changes how bad excess fat is for us.
不过 ，// 这些都改变不了多余脂肪对我们的伤害。

So, // how does our fat organ become so destructive // and can we help ourselves?
那么 ，// 我们的脂肪器官是如何变得如此具有破坏性的 // 我们能帮助自己吗？

What is fat?
什么是脂肪？

If you consume more energy than you burn, // you store it as triglycerides, // an organic battery bustling with energy, // collected in a large drop of fat // inside a white fat cell.
如果你消耗的能量超过消耗的能量，// 你会将其储存为甘油三酯，// 一个充满能量的有机电池，// 收集在一大滴脂肪中 // 置于白色脂肪细胞内。

Gain weight // and white fat cells expand with fat, // lose weight, // and they shrink.
体重增加 // 白细胞会随着脂肪一起膨胀，// 减肥，// 它们会缩小 。

Only in the last few decades // have we learned fat's most important job.
直到近几十年 ，我们才了解到脂肪最重要的作用。

Fat is an endocrine organ, // part of the system that makes and regulates your hormones, // chemical signals for your brain, // liver, // muscles, // digestive tract, // and immune system, // making them work correctly together.
脂肪是一种内分泌器官 ，是制造和调节激素的系统的一部分，负责大脑、肝脏、肌肉、消化道和免疫系统，使它们正常协同工作。

Unfortunately, // if you become overweight and obese, // your fat organ and the many hormones it releases turn insane.
不幸的是 ，如果你变得超重和肥胖，你的脂肪器官和它释放的多种激素会变得疯狂 。

In adults, // fat comes in two types of white fat depots.
在成年人中，// 脂肪分为两种白脂储备。

Most of it is subcutaneous fat, // the gooey soft stuff under your skin // that insulates against the cold // and serves as energy storage.
大部分是皮下脂肪 ，// 皮肤下那种软软黏稠的东西 // 能隔绝寒冷 // 并起到能量储存的作用。

The other one is visceral fat // nestled between your organs, // providing a soft cushion for your sensitive insides.
另一种是内脏脂肪 ， 夹在器官之间 ，为你敏感的内部提供柔软的缓冲。

But it also happens to be more dangerous.
但这也更危险。

These fat cells are super sensitive to stress hormones // like cortisol or adrenaline.
这些脂肪细胞对压力激素非常敏感 ，比如皮质醇或肾上腺素。

When they pick up a surge of stress, // they release fatty acids directly into your blood // and are picked up by your liver // as a rapid source of energy.
当它们感受到压力激增时，// 它们会直接释放脂肪酸到血液中 // 然后被肝脏接收 // 作为快速的能量来源。

On top of that, // your visceral fat is very metabolically active, // in a constant hormonal dialogue with the rest of your body.
此外 ，// 你的内脏脂肪代谢非常活跃 ，// 与身体其他部位持续进行激素对话。

This is also why the health of two people with the same weight and amount of fat // can be totally different.
这也是为什么两个体重和脂肪量相同的人的健康状况可能完全不同。

If you're pear-shaped // and your fat is mostly in your hips or limbs, // you're much less at risk // than someone apple-shaped // with a lot of fat in their torso.
如果你是梨形 ，脂肪主要集中在臀部或四肢 ，风险远低于苹果形且躯干脂肪很多的人。

Bite by bite.
一口一口地。

As you gain unhealthy weight, // excess visceral fat triggers a cascade of negative changes all at once.
随着体重不健康增加，// 过多的内脏脂肪会一次性引发一连串负面变化。

Fat cells bloat up to their limit // until they outgrow their blood supply // and get starved of oxygen.
脂肪细胞会膨胀到极限 // 直到它们的血液供应耗尽 // 导致缺氧。

They become critically stressed or even die. // This is bad.
他们会陷入极度压力，甚至死亡。这很糟糕。

If you're overweight or obese, // your fat is basically a wounded organ // leaking stress and poison into your system.
如果你超重或肥胖，// 你的脂肪基本上是一个受伤的器官 // 正在向你的系统泄漏压力和毒素。

Especially your visceral fat, // even more triggered by the extra stress hormones, // makes your blood more fatty.
尤其是内脏脂肪，// 额外的压力激素会触发更多，// 让你的血液更油腻。

This overfeeds your organs like your liver or your muscles, // who can't keep up anymore // and begin to take damage.
这会过度滋养你的肝脏或肌肉，// 它们无法再跟上 // 并开始受损 。

The cellular stress and dead fat cells are emergency signals // that call your immune system to a fight.
细胞压力和死亡脂肪细胞是紧急信号 ，召唤你的免疫系统战斗。

Armies of macrophages invade your fat tissue, // creating clusters trying to eliminate the cause of the stress, // but can't.
大量巨噬细胞入侵你的脂肪组织，// 形成簇状结构，试图消除压力的原因，// 但却做不到。

So they stay and call for more help.
于是他们留下来呼救。

In a lean person's fat // immune cells make up about 5% of cells.
瘦人脂肪中，免疫细胞大约占细胞的 5%。

In an obese person's fat // up to 40%.
肥胖者的脂肪含量可达40%。

Active immune cells cause inflammation // bloating up your tissue // and releasing even more alarm signals.
活跃的免疫细胞会引起炎症 // 膨胀你的组织 // 并释放更多警报信号。

Which is good for a short amount of time when you're sick // but if it becomes chronic // it's like your entire body is under friendly fire.
这在生病时短暂时是好事 // 但如果变成慢性病 // 就像全身都遭受了友军误伤 。

The inflammation molecules and fatty acids // rip countless tiny wounds // into the insides of your blood vessels // which leads to plaques // that desperately try to close them.
炎症分子和脂肪酸会撕裂无数微小的伤口，深入血管内部，形成拼命试图闭合的斑块 。

This narrows your blood vessels // and reduces the flow of oxygen-rich blood.
这会收缩你的血管 // 并减少富含氧气的血液流动。

Inflammation also causes your blood pressure to rise // so your heart has to work harder // and your risk of heart attacks or strokes // increases massively.
炎症还会导致血压升高 // 所以心脏必须更努力工作 // 心脏病发作或中风的风险大幅增加 //

To make things worse // your fat organ's hormone production gets out of whack.
更糟的是，你的脂肪器官激素分泌失调了。

Like leptin, // the satiety hormone.
比如瘦素 ，// 饱腹激素。

With a healthy amount of fat // leptin tells your brain when you have enough energy storage, // can eat less // and spend more energy.
摄入健康脂肪时，瘦素会告诉大脑你有足够的能量储存，可以吃得更少，消耗更多能量。

But if you have too much fat, // instead of becoming less hungry, // your brain becomes resistant against the constant flood of leptin.
但如果你脂肪过多，// 非但不会减少饥饿感，// 你的大脑反而会对持续的瘦素洪流产生抵抗力。

This breaks your internal food thermostat, // which is one of the reasons why many people who are overweight // feel intense hunger.
这会破坏你体内的食物恒温器 ，// 这也是许多超重者感到强烈饥饿的原因之一。

Their fat is screaming at their brain that they have enough, // but the brain is not hearing the message anymore.
他们的脂肪在大脑里大声喊着他们已经够了，// 但大脑已经听不到这个信息了。

Your sex hormones also get out of whack.
你的性激素也会失衡 。

Testosterone is lowered // while estrogen is overproduced.
睾酮降低 // 而雌激素过度产生。

In women // this increases the risk of breast cancer significantly.
女性// 这会显著增加乳腺癌的风险。

Most people are not aware // how much the risk of cancer rises with excess fat.
大多数人并不知道// 过多脂肪会增加癌症风险有多大。

In the US // almost 10% of all cancers are directly related to being overweight or obese.
在美国，// 几乎 10%的癌症与超重或肥胖直接相关 。

And to top this off, // obese cancer patients have much worse outcomes // succumbing to the disease more often and sooner.
更糟的是，// 肥胖癌症患者的结局更差 // 更容易、更早死于疾病。

None of this is great, // but even worse is what happens to one of your most important hormones, // insulin.
这些都不好 ，// 但更糟的是你最重要的激素之一—— 胰岛素会发生什么变化。

When the Hormone World Explodes
当激素世界爆发时

Insulin is a hormone produced by your pancreas // that tells your cells to open their tiny mouths // and eat up glucose from your blood.
胰岛素是胰腺产生的一种激素，// 它会告诉细胞张开小嘴 // 并从血液中吸收葡萄糖 。

It's your body's way of screaming, // "dinner time".
这是你身体在大声喊叫，// “吃饭时间到了”。

Disastrously bombarded by the stress from your excess fat, // cells all over your body become insulin resistant, // worse at eating glucose and taking up energy.
在多余脂肪带来的压力下，// 全身的细胞变得胰岛素抵抗 ，// 在摄入葡萄糖和消耗能量方面变得更差。

Your body tries to compensate // by pumping out more insulin, // screaming louder.
你的身体试图通过大量分泌胰岛素来补偿 ，// 发出更大声的尖叫声。

This can go on silently for many years // and progress to pre-diabetes, // with no or only subtle symptoms like fatigue or hunger.
这种情况可能无声持续多年 // 并发展为糖尿病前期，// 没有或只有轻微症状，比如疲劳或饥饿。

But as the damage accumulates, // eventually your body just can't keep up anymore.
但随着损伤积累 ，// 最终你的身体就跟不上了。

Something breaks // and you get type 2 diabetes.
一旦某样东西坏了 // 你就得了 2 型糖尿病 。

The cells responsible for insulin production are so overworked // that they stop functioning properly // and eventually give in.
负责胰岛素生成的细胞过度劳累 // 以至于功能停止 // 最终崩溃 。

Your insulin crashes drastically // and your body can't compensate.
你的胰岛素会剧烈下降 // 身体无法补偿。

While the blood is now saturated with glucose, // yet you're starving // and feel exhausted and unwell.
当血液中葡萄糖已经饱和时，// 你却饥饿 // 感到疲惫和不适。

Imagine this as trillions of tiny sharp shards // floating through you, // damaging your blood vessels, // nerves // and organs, // causing slow but constant damage everywhere.
想象成数万亿颗微小锋利的碎片 // 漂浮在你体内，// 损伤你的血管、神经 // 和器官，// 造成缓慢但持续的伤害。

At this point, // your body is so chronically inflamed // that almost all organ systems are affected.
此时 ，// 你的身体已经严重发炎 // 几乎所有器官系统都会受到影响。

The kidneys are overwhelmed, // making you pee way more.
肾脏过度受损，// 导致尿量大幅增加。

Your vision gets blurry, // your immune system is severely weakened, // wounds heal slower, // dying nerves lead to numbness and pain.
你的视力会模糊，// 免疫系统严重衰弱，// 伤口愈合变慢，// 神经死亡会导致麻木和疼痛。

You may experience shortness of breath, // chest discomfort, // erectile dysfunction // and high blood pressure, // problems with your memory, // focus, // mood // and even depression.
你可能会出现呼吸急促 、胸部不适、勃起功能障碍 、高血压、记忆力、注意力、情绪和抑郁问题。

Your risk of developing just about every possible deadly disease // goes through the roof.
你患几乎所有可能致命疾病的风险都会飙升。

On average, // type 2 diabetes shaves 10 years off your life // and reduces your health span massively, // arguably as much as smoking.
平均而言 ，// 2 型糖尿病能让你的寿命缩短 10 年 // 并大幅缩短健康寿命 ，// 可以说和吸烟一样严重。

If current obesity trends continue, // up to one in three Americans will have diabetes by 2050.
如果当前肥胖趋势持续，// 到 2050 年， 多达三分之一的美国人将患有糖尿病。

There is no nice way to put this.
这话说得不好听。

Excessive fat strains nearly every organ system in your body, // ages you much quicker, // and often leads to multi system damage and dysfunction.
过多的脂肪会让你身体几乎所有器官系统都紧张 ，// 让你衰老得更快，// 而且常常导致多系统损伤和功能障碍。

And yet // fat is still mostly discussed through the lens of aesthetics first // and health second.
然而 ，肥胖仍然是主要从审美和健康为先的角度来讨论。

Which is kind of baffling // considering that most // if not all // of the toxic effects of your excess fat // basically go away // as soon as you lose it // and start eating a healthy-ish diet.
这有点令人费解 ，因为大多数多余脂肪的毒性影响基本上都会消失 ， 一旦你减掉脂肪，开始吃比较健康的饮食。

Once your fat cells contract again, // they stop being stressed // and your immune system calms down.
一旦脂肪细胞再次收缩 ，// 它们就不再紧张 // 免疫系统也会平静下来。

The excess blood fat and sugar drops to normal levels // and your body recovers.
多余的血液脂肪和血糖会降到正常水平// 你的身体恢复 。

Even if you already have full-blown diabetes type 2, // by losing weight // you can reverse many of the negative effects // and drastically improve your health and lifespan.
即使你已经患有严重的 2 型糖尿病，// 通过减肥 // 你也能逆转许多负面影响 // 并大幅改善你的健康和寿命。

So if you've been waiting for a push to get started, // now is the time.
所以如果你一直在等待一个推动开始，// 现在就是时候了。
